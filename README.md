## 環境構築
Docker環境を起動しておいてください（DockerDesktop, Orbstack, etc...）
```bash
# レポジトリをクローン
git clone git@github.com:RentoYabuki06/ft_transcendence.git

# コンテナ起動
docker compose up
```

- フロントエンド： http://localhost:5173
- バックエンド：http://localhost:3000
