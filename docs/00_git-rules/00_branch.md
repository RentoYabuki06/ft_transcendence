## ブランチの切り方

```bash
git checkout -b <prefix>/<issue-no>-<title>
```

### prefix とは

行う編集の分類(概要)を示すもの

|番号|名称|説明|
|:-:|:--|:--|
|0|feat|機能追加|
|1|fix|機能修正(機能を変更するときなど))|
|2|bugfix|バグ修正|
|3|delete|機能削除|
|4|test|テスト関係|
|5|docs|説明関係|

<br>

### issue-no とは

ISSUEの番号

<br>

### title とは

ブランチをパット見たときに判断することができる名称
